# React Native Expo App

React Native Image Gallery App using SQLite as database

## Features:

```python
    1. Take an image.
    2. View taken images.
    3. Update image name.
    4. Delete image.
    5. Integrate map
```

## Installation

npm install

```bash
Run project:
npm run dev
```
